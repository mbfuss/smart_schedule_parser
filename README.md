# smart_schedule_parser

**smart_schedule_parser** — сервис для автоматического скачивания, парсинга и структурирования расписаний из PDF-файлов для образовательных учреждений.

## Возможности

- Скачивание PDF-файлов расписаний с заданного сайтаА.
- Парсинг PDF-файлов и извлечение информации о корпусах, институтах, формах обучения и группах.
- Структурирование данных в удобном виде для дальнейшей обработки.
- Логирование и обработка ошибок.
- Возможность запуска в Docker-контейнере.

## Структура проекта

- `cmd/` — точка входа, основной исполняемый файл.
- `internal/` — основная бизнес-логика (парсер, кравлер, провайдеры, ресурсы, сервер и т.д.).
- `pkg/` — вспомогательные пакеты и утилиты.
- `pdf_list/` — директория для хранения скачанных и распарсенных PDF-файлов.
- `docker/` — Dockerfile и docker-compose для контейнеризации.
- `scripts/` — вспомогательные скрипты (например, для конвертации PDF в CSV).
- `test/` — тестовые данные и тесты.я
- `docs/` — документация и схемы.

## Запуск приложения:

### Production среда:
``
Требования: docker-compose.
``
1. Клонируйте репозиторий:
``
git clone git@github.com:mbfuss/smart_schedule_parser.git
``

2. Выполните запуск сервиса в Docker-контейнере:
``
make docker-up
``
### Для разработки:
``
Требования: go version 1.25, python 3.10.
``
1. Клонируйте репозиторий:
2. 
   ``
   git clone git@github.com:mbfuss/smart_schedule_parser.git
   ``

3. Выполните установку зависимостей для go и python.

`Python: pandas, pdfplumber`

`GO: зависимости должны автоматически загрузиться при выполнении команды запуска приложения`

4. Выполните команду запуска приложения:
`make run`

## Информация по API:
1. `getschedule?urlSchedule=<url_страницы_с_расписанием >`

_Например: http://localhost:8080/getschedule?urlSchedule=www.vavilovsar.ru/ucheba/raspisanie-zanyatii_

## Дополнительная информация
1. PDF файлы скачиваются и хранятся в корне репозитория директории `pdf_list`.
2. Логи хранятся в директории `log`  в корне репозитория в директории `log`.
