# Smart Schedule Bot (Telegram) by [r3ynD](https://github.com/r3ynD)

Телеграм-бот для просмотра расписания по **группе**, **преподавателю** и **аудитории**, с поддержкой **числителя/знаменателя** и генерацией **картинки недели (PNG)**.

---

## Что умеет бот

- Показывать расписание группы:
  - **Сегодня**
  - **Завтра**
  - **Неделя (текстом)**
  - **Неделя (картинкой PNG)**
- Поиск расписания:
  - по **преподавателю** (фамилия → выбор → неделя)
  - по **аудитории**:
    - выбрать **УК** кнопкой или “Любой УК”
    - вводить **аудиторию** `401` или **УК/ауд** `2/401`
- Подсказки “похожие значения”, если не найдено:
  - группы
  - преподаватели
  - аудитории
- Админ-команды (под паролем):
  - `/update` — вручную обновить snapshot
  - `/status` — посмотреть статус обновлений
- Горячая перезагрузка snapshot:
  - если файл snapshot изменился → репозиторий перечитывается, кеш чистится, старые PNG удаляются

---

## Стек и зависимости

- **Node.js** (желательно LTS)
- **telegraf** — Telegram Bot API (команды, кнопки, обработчики)
- **lru-cache** — кеш ответов “сегодня/завтра/неделя” на короткое время
- **node:fs** — чтение snapshot/status и отслеживание изменений
- Внутренние модули проекта:
  - `SnapshotRepo` — загрузка и индексация snapshot (группы/преподы/аудитории/УК)
  - `renderer` — генерация PNG расписания недели + очистка старых версий
  - `db` — хранение состояния пользователя (группа, режим недели и т.п.)
  - `updater` — обновление snapshot (ручной триггер `/update`)

## Установка и запуск

### 1) Установка npm-пакетов

```bash
npm install
```

### 2) Главное меню (inline-кнопки)

Сегодня → расписание группы на сегодня

Завтра → расписание группы на завтра

Неделя → расписание группы на неделю (текст)

🖼 Картинка недели → генерация PNG расписания недели

Преподаватель → ввод фамилии → выбор из списка → неделя

Аудитория → выбор УК (или “Любой УК”) → ввод аудитории → неделя

Сменить группу → переводит в режим ввода группы

### Формат поиска аудитории

Поддерживается ввод:

401 — искать аудиторию по всем УК

2/401 — искать аудиторию в конкретном УК

Парсится утилитой parseUkRoom():

если есть / → ukNum + room

иначе → room без ukNum

Если не найдено — выдаются “Похожие” (suggestions).

### 4) Архитектура: кто за что отвечает
Bot module (файл с createBot)

Отвечает за:

обработчики команд /start, /update, /status, /cancel

обработчики кнопок bot.action(...)

горячую перезагрузку snapshot + очистку старых PNG
